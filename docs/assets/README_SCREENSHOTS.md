# README 스크린샷 체크리스트

README에 실제 사용 예시를 추가할 때는 아래 synthetic demo 파일만 사용합니다.

- `docs/samples/sample_course.pdf`
- `docs/samples/sample_lecture.vtt`

개인 PDF, 유료 학습자료, 복사한 자막 원문, 개인 Obsidian vault 경로, 로컬 PC 경로가 보이는 화면은 캡처하지 않습니다.

## 반드시 찍을 화면

1. PDF2Obsidian GUI에 `sample_course.pdf`, `sample_lecture.vtt`를 추가한 화면
2. 변환 완료 후 생성된 output 폴더 화면
3. Obsidian에서 `sample_course.md`를 연 화면
4. Obsidian에서 `sample_lecture.md`를 연 화면
5. 원본 PDF Before / 변환된 Markdown After 비교 화면

## 추천 저장 파일명

- `gui-sample-files-loaded.png`
- `output-folder-demo.png`
- `obsidian-sample-course.png`
- `obsidian-sample-lecture.png`
- `pdf-before-markdown-after.png`

## 캡처할 때 주의할 점

- 화면에 실제 개인 폴더 경로가 크게 보이면 안 됩니다.
- 개인 자료명, 유료 강의명, 실제 자막 내용이 보이면 안 됩니다.
- README에 넣을 이미지는 가능하면 가로 1200px 이상으로 캡처합니다.
- Obsidian 화면은 demo 파일만 열린 상태에서 캡처합니다.
- 변환 결과 폴더도 demo 파일 결과만 보이게 정리한 뒤 캡처합니다.

## 캡처 파일을 넣을 위치

캡처가 끝나면 아래 경로에 이미지를 넣습니다.

```text
docs/assets/
├─ gui-sample-files-loaded.png
├─ output-folder-demo.png
├─ obsidian-sample-course.png
├─ obsidian-sample-lecture.png
└─ pdf-before-markdown-after.png
```
