# README Screenshot Checklist

Use only the synthetic demo files in this repository when creating screenshots:

- `docs/samples/sample_course.pdf`
- `docs/samples/sample_lecture.vtt`

Do not capture private PDFs, paid learning material, copied subtitle text, personal Obsidian vault paths, or local machine paths.

## Required Screenshots

1. GUI with `sample_course.pdf` and `sample_lecture.vtt` added to the file list.
2. Output folder after conversion completes.
3. `sample_course.md` opened in Obsidian.
4. `sample_lecture.md` opened in Obsidian.
5. PDF Before / Markdown After comparison view.

## Recommended File Names

- `gui-sample-files-loaded.png`
- `output-folder-demo.png`
- `obsidian-sample-course.png`
- `obsidian-sample-lecture.png`
- `pdf-before-markdown-after.png`
